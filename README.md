EN

Our first larger Java project. We simulated a university data entry scenario through plain Java code - entering and storing data about students, professors, and assistants in separate lists, and printing them based on different commands entered in the console.

What I practiced:

Building a menu-driven console application using a while loop

Working with multiple ArrayList collections simultaneously

Using inheritance - Asistent extends Profesor, which extends Osoba

Extracting reusable logic into helper methods (vratiOsobu, vratiProfesora)

Handling user input validation and unknown commands

Managing multiple object types with shared base class properties

SR

Naš prvi veći Java projekat koji simulira sistem za unos podataka na fakultetu putem konzole. Program omogućava korisniku da dodaje studente, profesore i asistente, čuva ih u zasebnim listama i prikazuje ih pojedinačno ili sve zajedno kroz jednostavan meni baziran na komandama.

Šta sam vežbao:

Pravljenje konzolne aplikacije sa menijem korišćenjem while petlje

Rad sa više ArrayList kolekcija istovremeno

Korišćenje nasleđivanja - Asistent proširuje Profesor, koji proširuje Osoba

Izdvajanje logike koja se ponavlja u pomoćne metode (vratiOsobu, vratiProfesora)

Rukovanje validacijom korisničkog unosa i nepoznatim komandama

Upravljanje više tipova objekata sa svojstvima zajedničke osnovne klase
